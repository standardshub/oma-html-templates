Test Case Id

LwM2M-1.0-con-DMSE-001

Test Objecta(DUT)

LwM2M Client device under test

Test Case Description

(Test Purpose)

Verify that the DUT(LwM2M Client) can respond properly to the ‘READ’ message from the Tester(LwM2M Server). For all of the readable mandatory objects(LwM2M Server and Device)  

Specification Reference

Section 5.4.1

SCR Reference

LwM2M-DMSE-001-C-M

Tool

LwM2M Conformance Tester_TTA_v1.0

Test code

ACK-2.05   MID=value, Token=value, OptionSet={"Content-Format":"application/vnd.oma.lwm2m+tlv"}, payload

Preconditions
  • Equipment:
    • 1 LwM2M client(DUT) / 1 LwM2M server(Tester)
  • State:
    • The DUT has the Objects and Object Instances(Mandatory and Optional).
    • The DUT and Tester support ‘tlv’ content format.
    • The DUT is registered to the Tester with No SEC mode.
  • Continuation of / Can be tested at the same time as:
    • LwM2M-1.0-con-CR-001 – Verify that the DUT can be registered with a message including URI-Path and Query Parameters (endpoint name, lifetime)
  • Prerequisite for this test:
    • None
  • Applicability:
    • ICS
      - 5-1. Support of “Read” operation
    • IXIT 
      - Object Instances of Mandatory Objects(LwM2M Server and   
       Device)
      - Endpoint name, lifetime
      - IP Address, Port Number

Test Procedure
  1. The Tester send ‘READ’ message(CoAP/GET) to the DUT.
    (GET /1/, LwM2M Server)
  2. The DUT send response message(2.05 Content).
    (The Object Instance(s) correspond to the IXIT)
    < ACK-2.05   MID=value, Token=value, OptionSet={"Content-Format":"application/vnd.oma.lwm2m+tlv"}, payload>
  3. The Tester send ‘READ’ message(CoAP/GET) to the DUT.
    (GET /3/, Device)
  4. The DUT send response message(2.05 Content).
    (The Object Instance(s) correspond to the IXIT).
    < ACK-2.05   MID=value, Token=value, OptionSet={"Content-Format":"application/vnd.oma.lwm2m+tlv"}, payload>

Pass-Criteria

<Verify that the DUT properly returned the object instances to the Tester.

  1. The DUT returned all of the object instance(s) of LwM2M Server object.
  2. The DUT returned all of the object instance(s) of Device object.
